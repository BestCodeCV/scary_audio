Política de Privacidad

Última actualización: 30 de Octubre de 2025

1. Recopilación de Información

Scary Audio 2025 (en adelante, "la Aplicación") es propiedad y está operada por StudioCV. Nuestra Aplicación está diseñada para la reproducción de sonidos internos temáticos.

La Aplicación NO recopila, almacena, ni transmite ninguna información de identificación personal (IIP) de sus usuarios. Todos los sonidos se almacenan internamente dentro de la Aplicación y no se accede a la memoria externa del dispositivo para leer archivos personales.

2. Uso de Permisos

La Aplicación solicita permisos específicos del sistema para ejecutar las siguientes funciones:

  • Modificar la Configuración del Sistema (WRITE_SETTINGS): Este permiso se utiliza exclusivamente para permitir al usuario **establecer los sonidos de terror internos como Tono de Llamada (Ringtone) o Tono de Alarma**. La Aplicación no realiza ninguna otra modificación en el sistema.
  • Almacenamiento (WRITE/READ EXTERNAL STORAGE - según la versión de Android): Necesario únicamente para la funcionalidad de **Compartir Audio**, ya que el archivo de sonido debe ser copiado temporalmente al almacenamiento cache del dispositivo para ser compartido a través de otras aplicaciones.

3. Servicios de Terceros (Publicidad)

La Aplicación utiliza el servicio de publicidad **Appodeal** para generar ingresos. Appodeal puede recopilar información no personal (como identificadores de dispositivos, ubicación aproximada e información de uso) para mostrar anuncios relevantes. Le recomendamos revisar la política de privacidad de Appodeal para obtener más detalles sobre su manejo de datos.
