Política de Eliminación de Datos de Usuario

Esta política detalla cómo los usuarios pueden solicitar la eliminación de los datos recopilados por la aplicación **Scary Sounds 2025**.

Datos Recopilados y Compartidos

La aplicación **Scary Sounds 2025** no requiere la creación de cuentas de usuario ni recopila información personal identificable como nombres, correos electrónicos o números de teléfono. Los únicos datos de usuario persistentes recopilados y compartidos son:

  • El **ID de Publicidad (Advertising ID)** del dispositivo.
  • Datos de uso y rendimiento de la aplicación a través de servicios de analítica.

Estos datos se utilizan exclusivamente para la analítica de rendimiento de la aplicación y para la personalización y segmentación de anuncios a través de terceros (Google AdMob).

Procedimiento para la Eliminación de Datos

Dado que la aplicación no almacena datos de cuenta personales en sus propios servidores, la eliminación de los identificadores de rastreo se gestiona a través de los controles de privacidad de Google y Android.

1. Eliminación y Restablecimiento del ID de Publicidad (Advertising ID)

El usuario puede eliminar o restablecer completamente su ID de Publicidad de Android en cualquier momento, lo que impide que las redes de publicidad sigan rastreando y asociando datos a ese identificador.

Pasos a seguir en el dispositivo Android:
  1. Ir a **Ajustes** (Settings) del dispositivo.
  2. Buscar la sección **Privacidad** (Privacy) o **Google**.
  3. Seleccionar **Anuncios** (Ads).
  4. Seleccionar **Eliminar ID de Publicidad** (Delete Advertising ID) o **Restablecer ID de Publicidad** (Reset Advertising ID).

2. Eliminación de Datos de Uso (Analítica)

Los datos anónimos de uso que se recopilan están asociados a un identificador de instalación y se retienen según las políticas de retención de Google Analytics/Firebase.

Para la eliminación completa de datos vinculados a la aplicación, el usuario debe desinstalar la aplicación y luego borrar la caché y los datos de la aplicación en la configuración del sistema operativo (Ajustes > Aplicaciones > Scary Sounds 2025 > Almacenamiento > Borrar datos y caché).
